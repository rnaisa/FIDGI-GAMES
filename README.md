# FIDGI-GAMES

A collection of four fidgeting-friendly minigames.

## How To Use

[Here](https://rnaisa.github.io/FIDGI-GAMES/) is a demo. Make sure to allow sound to be played by the website, as it won't be playing before interacting with it first.
